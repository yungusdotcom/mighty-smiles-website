# Mighty Smiles Dentistry - Website

Patient-facing website for Mighty Smiles Dentistry, North Las Vegas, NV.

**Repo:** https://github.com/yungusdotcom/mighty-smiles-website

## Quick Start

git clone, cd, npx serve public. Push to main = auto-deploy via Vercel.

## Structure

public/index.html - Full site (deployed)
assets/ - Source images and logos (local ref)
docs/DEVELOPMENT.md - Build history and roadmap

## Architecture

Single-file static HTML. All CSS inline, images base64, SVG inlined. Playfair Display + Outfit fonts via CDN.
